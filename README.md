# Profiler

## Install

composer project akeb/profiler

Composer config

```json
{
    "require": {
        "akeb/profiler": "^1.1.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/profiler"
        }
    ]
}
```

or

```bash
composer require akeb/profiler
```
