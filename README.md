## Install

composer project akeb/profiler

Composer config

```json
{
    "require": {
        "akeb/profiler": "^1.0.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://gitlab.pvt/gapa/mrgs/profiler"
        }
    ]
}
```

or

```json
{
    "require": {
```
